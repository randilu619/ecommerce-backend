# Automated E-Commerce Backend Deployment Pipeline

This repository hosts the production deployment workflow and configuration for our containerized E-Commerce Backend web application deployed securely on AWS using modern DevOps methodologies.

## Architectural Features
* **Containerization:** Docker multi-stage builds.
* **CI/CD Pipeline:** Automated build-and-ship workflow using GitHub Actions.
* **Registry Management:** AWS Elastic Container Registry (ECR).
* **High Availability & Scalability:** AWS Application Load Balancer (ALB) & AWS Auto Scaling Groups (ASG).
* **Compute Host:** AWS EC2 Linux (Ubuntu) environment.
